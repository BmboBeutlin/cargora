# Cargora

> Browser-Multiplayer-Logistikspiel mit modularer Tiefe. Hobby-Projekt.

**Status:** Pre-Alpha. Stunde-1-Prototyp seit 2026-05-03.

---

## Vision

Du verwaltest eine Logistik-Firma in einer simulierten Welt — jeder LKW, jeder Reifen, jeder Flughafen, jede Fabrik ist konfigurierbar. Echte Märkte, echte Wechselwirkungen. Inspiriert von OpenTTD, Factorio und SimuTrans Extended, aber mit Tiefe statt Breite.

Detaillierte Vision: [`docs/GAME_DESIGN.md`](docs/GAME_DESIGN.md)

---

## Quickstart

```bash
git clone https://github.com/BmboBeutlin/cargora.git
cd cargora
npm install
npm run dev
```

Browser öffnet `http://localhost:5173` automatisch.
